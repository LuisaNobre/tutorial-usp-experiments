    SGPF - Sistema de Gestao de Pesquisa Florestal

    Documentacao do sistema

    Este portal reune tutoriais para apoiar o cadastro, a atualizacao e a consulta de informacoes de experimentos no SGPF, com instrucoes objetivas, imagens e videos.

    Sobre esta documentacao

    Aqui voce encontra orientacoes para:

    • iniciar o uso do sistema;
    • cadastrar novos experimentos;
    • complementar listas e caracteristicas usadas no preenchimento;
    • atualizar informacoes de experimentos ja cadastrados;
    • exportar resultados para consulta e analise.

    Sequencia recomendada

    Se esta for sua primeira vez no sistema, a ordem mais indicada e:

    1. Primeiros Passos
    2. Criar experimento
    3. Complementar Caracteristicas
    4. Atualizar dados de experimento
    5. Exportar resultados
