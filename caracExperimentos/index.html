    Complementar características

    Nesta página reunimos mini tutoriais para os casos em que uma opção necessária não aparece na lista de uma variável categórica do sistema.

    Isso pode acontecer, por exemplo, durante o cadastro de experimento, quando o usuário precisa selecionar um valor em um campo como Programa, mas a opção desejada ainda não está cadastrada.

    Nesses casos, é necessário adicionar a nova opção na lista correspondente antes de concluir o preenchimento do experimento.

    Cada seção abaixo apresenta um tutorial curto e específico para mostrar como incluir novos itens em uma lista do sistema.

    Programa

    Use esta seção quando for necessário adicionar uma nova opção na lista da variável Programa.

    Passo a passo

    1. No menu superior, clique em Tabelas, depois em Pesquisa e, em seguida, em Cadastro de Programas de Pesquisa.

    1. Esse caminho abrirá uma tabela com todos os programas cadastrados até o momento.

    2. Clique no botão com o símbolo de +, localizado acima da tabela, para adicionar um novo programa.

    3. Uma nova linha em branco será criada.

    4. A coluna Programa Id é preenchida automaticamente pelo sistema, portanto não deve ser preenchida pelo usuário.

    5. Preencha os campos Nome do programa e Descrição com as informações do novo programa.

    6. Ao final, clique no botão de check para salvar.
