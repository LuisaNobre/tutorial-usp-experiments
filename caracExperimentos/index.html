    Complementar Caracteristicas

    Nesta pagina reunimos mini tutoriais para os casos em que uma opcao necessaria nao aparece na lista de uma variavel categorica do sistema.

    Isso pode acontecer, por exemplo, durante o cadastro de experimento, quando o usuario precisa selecionar um valor em um campo como Programa, mas a opcao desejada ainda nao esta cadastrada.

    Nesses casos, e necessario adicionar a nova opcao na lista correspondente antes de concluir o preenchimento do experimento.

    Cada secao abaixo sera um tutorial curto e especifico para mostrar como incluir novos itens em uma lista do sistema.

    Como esta pagina esta organizada

    Cada mini tutorial tera a mesma estrutura:

    • objetivo do tutorial;
    • passo a passo com imagens;
    • observacoes importantes, quando necessario;
    • video demonstrando o procedimento.

    Enquanto os tutoriais ainda estiverem sendo montados, cada secao ficara preparada para receber o conteudo.

    Programa

    Use esta secao quando for necessario adicionar uma nova opcao na lista da variavel Programa.

    Passo a passo

    1. No menu superior, clique em Tabelas, depois em Pesquisa e, em seguida, em Cadastro de Programas de Pesquisa.

    1. Esse caminho abrira uma tabela com todos os programas cadastrados ate o momento.

    2. Clique no botao com o simbolo de +, localizado acima da tabela, para adicionar um novo programa.

    3. Uma nova linha em branco sera criada.

    4. A coluna Programa Id e preenchida automaticamente pelo sistema, portanto nao deve ser preenchida pelo usuario.

    5. Preencha os campos Nome do programa e Descricao com as informacoes do novo programa.

    6. Ao final, clique no botao de check para salvar.
