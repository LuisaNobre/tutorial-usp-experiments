## 📌 Passo a passo

1. Clique em **New Experiment**
2. Escolha o nome do experimento
3. Configure os parâmetros
4. Clique em **Save**

---

## ⚠️ Problemas comuns

- O botão "Save" não aparece → verifique campos obrigatórios  
- Erro ao criar → tente reiniciar o sistema

---

## 💡 Dica

Use nomes claros para facilitar a organização dos experimentos.
